## 2. 业务目标

**一句话（草稿）**：提供一套可 fork 的 AI 原生软件工厂方法论与目录约定，使新项目能按统一阶段产出文档与代码，并可用示例项目（monitorV2）演练全流程。

**可衡量结果（草稿）**：
1. Fork 后 1 天内完成 `00-project.md` + `01-business/` 最小闭环
2. 按阶段 README 可回答「角色 / 流程 / 功能 / 权限 / 页面 / 数据」六问
3. M3 前至少一条主流程可本地演示（示例项目接入时再量化 SLA）
