## 3. 🟡 P2 — 锦上添花(可选)

### P2-01 缺 Junction 自动重建脚本

**位置**:`14-scripts/`(建议新增 `setup-cursor.ps1`)

**作用**:新 clone 仓库后一键创建 `.cursor/prompts|skills|agents` 三个 junction,并校验 `.cursor/rules/` 存在。

**骨架**:

```powershell
# setup-cursor.ps1
$root = $PSScriptRoot | Split-Path -Parent
$targets = @('prompts', 'skills', 'agents')
foreach ($t in $targets) {
    $link = Join-Path $root ".cursor\$t"
    $src  = Join-Path $root "0$(([array]::IndexOf($targets,$t))+6)-$($t.TrimEnd('s'))s"
    if (Test-Path $link) { Write-Host "✅ $t OK"; continue }
    cmd /c mklink /J $link $src | Out-Null
    Write-Host "🔗 $t -> $src"
}
```

(注:真实脚本请按本仓目录编号写,如 `06-prompts`、`08-skills`、`09-agents`)
