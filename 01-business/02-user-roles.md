# 用户角色

> 演示虚构：DemoAlert。与 `05-permission-design.md` 角色编码对齐。

## 角色清单

### 角色1：系统管理员
- **角色标识**: `admin`
- **角色描述**: 管理用户、角色与系统配置；可查看全部组织数据（演示简化为「全部」）
- **使用场景**: 开通账号、调整角色、排查权限问题
- **权限范围**:
  - 用户与角色管理
  - 系统配置
  - 全部告警读写

### 角色2：值班操作员
- **角色标识**: `operator`
- **角色描述**: 日常认领与处置告警
- **使用场景**: 值班期间处理本组织告警
- **权限范围**:
  - 本组织告警查询 / 认领 / 关闭
  - 导出本组织告警列表

### 角色3：只读观察者
- **角色标识**: `viewer`
- **角色描述**: 查看告警与报表，不可改状态
- **使用场景**: 业务方关注进展
- **权限范围**:
  - 本组织告警只读

## 角色关系图
```
admin ──管理──> operator / viewer
operator / viewer ──归属──> 组织(Org)
```
