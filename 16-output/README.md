# 16 - Output（输出物）

构建产物、报告与日志的**暂存/归档**目录。默认被 `.gitignore` 忽略（保留 `**/.gitkeep`）；需要入库的终稿请显式评估后再取消忽略或改放到 `docs` 仓库策略允许的路径。

## 子目录与保留策略

| 目录 | 内容 | 建议保留期 | 清理 |
|------|------|------------|------|
| `builds/` | 安装包、镜像导出、制品包 | 最近 3 次成功构建 | 发布成功且远端制品库已有副本后可删本地 |
| `docs/` | 导出的 PDF/HTML 文档 | 与对应版本同寿命 | 版本废弃后归档或删除 |
| `reports/` | 测试/覆盖率/安全扫描报告 | ≥ 当前 Sprint + 1 | Sprint 结束后压缩归档 |
| `logs/` | 本地脚本/工具运行日志 | 7–14 天 | 定期清空；勿提交真实日志 |

## 约定

1. 不在此目录存放密钥、生产数据转储
2. 对外交付物优先登记到 `CHANGELOG.md` / 发版说明
3. CI 产物优先推制品库；本目录偏本地与临时归档
