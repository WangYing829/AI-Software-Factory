# API 版本策略

> **DemoAlert（S02）**

## 版本规则
- URL 前缀：`/api/v{major}` → 当前 **v1**
- 破坏性变更升 major；兼容新增在 v1 内演进
- OpenAPI `info.version` 跟随工厂/演示文档版本（当前 0.2.x）

## 兼容性承诺
| 变更类型 | 是否破坏 | 处理 |
|----------|----------|------|
| 新增字段/接口 | 否 | 更新 OpenAPI |
| 删字段/改状态机语义 | 是 | v2 或弃用期 |
| 改错误码 | 视情况 | 公告 |

## 当前
| API major | 状态 | 说明 |
|-----------|------|------|
| v1 | 演示可用 | DemoAlert 路径已写入 OpenAPI |
