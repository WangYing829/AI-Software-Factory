# 导入 / 导出 API

## 范围
| 能力 | 格式 | 权限 | 优先级 |
|------|------|------|--------|
| 导出 `[示例: 业务列表]` | CSV/XLSX | `rpt:export` | P1 |
| 导入 | CSV | `biz:entity:write` | P2 |

## 导出
- **方法/路径**: `GET /api/v1/biz/entities/export`
- **限制**: 最大行数 / 异步阈值说明
- **字段映射**: 与列表列一致

## 导入
- **方法/路径**: `POST /api/v1/biz/entities/import`
- **校验**: 模板列、必填、重复键策略
- **结果**: 成功数 / 失败行报告

## 验收
- [ ] 大文件策略（同步 vs 异步）已声明
- [ ] 失败可追溯
