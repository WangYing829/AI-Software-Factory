# Testing Skill

## When
测试计划、用例设计、自动化或缺陷跟踪时。

## Steps
1. 对照 `02-product/` 验收标准与 `05-api/`
2. 用例与计划遵循 `10-knowledge/07-testing-standards.md`
3. 自动化放 `13-src/tests/`
4. 缺陷记录到 `07-tasks/04-bug-tracking.md`

## Output
- 测试说明可写入阶段文档或 `16-output/reports/`
- Bug 状态与 Sprint 同步更新 `11-context/`
