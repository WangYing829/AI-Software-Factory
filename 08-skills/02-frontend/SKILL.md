# Frontend Skill

## When
页面、组件、前端联调、交互实现时。

## Steps
1. 读 `01-business/06-page-design.md`、`02-product/`（含 `ui/`）、`05-api/`
2. 按契约对接 API，避免臆造字段
3. 实现后同步必要的页面/UI 文档变更
4. 覆盖主流程与权限相关展示

## Output locations
- 代码：`13-src/` 前端模块
- 文档：`01-business/06-page-design.md`；补充材料 `02-product/ui/`
