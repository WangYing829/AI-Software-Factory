# Backend Skill

## When
实现服务端接口、领域逻辑、中间件集成时。

## Steps
1. 读 `11-context/` 与 `03-architecture/`、`05-api/`
2. 确认任务在 `07-tasks/03-todo.md` 或当前 Sprint
3. 按模块改 `13-src/`
4. 接口/表结构变更同步 `05-api/`、`04-database/`
5. 补充测试（`13-src/tests/`）

## Output locations
- 代码：`13-src/`
- 接口：`05-api/`
- 部署约束：见 `03-architecture/` + `10-knowledge/05-deployment-standards.md`
