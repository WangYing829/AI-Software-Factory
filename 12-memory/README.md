# 12 - Memory（项目记忆）

## 说明

长期记录项目中的关键信息，避免"重复踩坑"。

## 文件说明

| 文件 | 内容 |
|------|------|
| `01-project-memory.md` | 项目关键信息（配置、环境、依赖版本） |
| `02-tech-memory.md` | 技术选型理由、技术难点解决方案 |
| `03-bug-memory.md` | 典型 Bug 及解决方案 |
| `04-lessons-learned.md` | 经验教训 |
| `05-solutions-library.md` | 可复用的解决方案 |

## 记录格式

```markdown
## [日期] [标题]

**背景**: 
**问题**: 
**解决方案**: 
**关键代码/配置**: 
**参考链接**: 
**标签**: #bug #mysql #performance
```

## 使用方式

- 遇到问题时先搜索 Memory
- 解决问题后及时记录
- 定期整理和归档
- 作为新成员 onboarding 的参考资料
