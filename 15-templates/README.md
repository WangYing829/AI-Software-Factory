# 15 - Templates（模板库）

可复用的协作模板。复制到目标位置后填写，不要直接改空原件当「已完成」。

## 清单与使用时机

| 模板 | 使用时机 | 产出落点 |
|------|----------|----------|
| `pr-template.md` | 开 Pull Request | GitHub/GitLab PR 描述 |
| `issue-template.md` | 提需求 / Bug / 任务 | Issue 或 `07-tasks/` |
| `meeting-template.md` | 评审 / 站会 / 决策会 | `11-context/06-meeting-notes/` |
| `sprint-template.md` | 开 Sprint 计划 | `07-tasks/01-sprint-planning.md` 或 `sprints/` |
| `release-checklist.md` | 打 tag / 发版前 | 发版记录 / `16-output/docs/` |

## 子目录

| 路径 | 用途 |
|------|------|
| `assets/` | 模板配图、示意图素材 |
| `config/` | 配置样例（无密钥） |

## 约定

- 模板至少保留标题与必填章节；禁止提交 0 字节模板
- 与 `10-knowledge/06-code-review-checklist.md`、`CHANGELOG.md` 交叉引用保持一致
