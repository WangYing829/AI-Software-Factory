# 安全设计

> **DemoAlert（S02）**

## 认证与授权
- 认证：JWT（示例）或 Session；文档默认 Bearer Token
- 授权：RBAC + `org_id` 数据范围；对齐 `01-business/05-permission-design.md`
- 默认拒绝

## 传输与存储
| 项 | 要求 |
|----|------|
| TLS | 生产强制 |
| 密钥 | 环境注入 |
| 密码 | 哈希存储；日志脱敏 |

## 威胁与缓解

| 威胁 | 缓解 |
|------|------|
| 越权访问他组织告警 | 查询强制 org 过滤（非 admin） |
| 重复认领竞态 | 条件更新 `status=open` |
| 注入 | 预编译 / 参数绑定 |

## 审计
- 登录失败、认领、关闭写入 `alert_event` 或审计日志

## 验收
- [x] P0 接口鉴权说明齐全
- [x] 管理端口不公网暴露
