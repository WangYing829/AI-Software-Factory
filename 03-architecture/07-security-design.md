## 传输与存储
| 项 | 要求 |
|----|------|
| TLS | 生产强制 |
| 密钥 | 环境注入，不入库 |
| 敏感字段 | 日志脱敏 |

## 威胁与缓解（骨架）

| 威胁 | 缓解 |
|------|------|
| 越权访问 | 权限点 + 数据范围 |
| 注入 | 参数绑定 / 预编译 |
| XSS | 前端转义 + CSP（按需） |
