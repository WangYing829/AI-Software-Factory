# 模块设计

## 模块清单

| 模块 | 职责 | 对内依赖 | 对外接口 | Owner |
|------|------|----------|----------|-------|
| `auth` | 登录鉴权 | — | Token API | |
| `biz` | `[示例: 核心业务]` | auth | REST | |
| `admin` | 系统配置 | auth | REST | |
| `common` | 统一返回/异常/工具 | — | — | |

## 包 / 目录建议（后端）

```
.../auth/
.../biz/
.../admin/
.../common/
```

## 模块交互

```
auth ← biz / admin
biz → DB / Redis
```
