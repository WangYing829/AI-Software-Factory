## US-003 详述

**故事**: 作为 operator，我想认领一条 open 告警，以便团队知道由我处理。

**验收标准**:
1. Given 告警状态为 open 且属本组织 When 点击认领 Then 状态变为 acked 且处理人为当前用户
2. Given 告警已被他人认领 When 再次认领 Then 失败并提示当前处理人
3. Given 角色为 viewer When 打开详情 Then 无认领按钮且 API 返回 403

**备注 / UI**: 页面 `P-DETAIL`

## US-004 详述

**故事**: 作为 operator，我想关闭已认领告警并填写原因，以便留下可审计记录。

**验收标准**:
1. Given 状态为 acked When 选择原因并关闭 Then 状态 closed 且时间线新增关闭事件
2. Given 原因为空 When 提交 Then 校验失败且状态仍为 acked
3. Given viewer When 调用关闭接口 Then 403
